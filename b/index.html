Operační systémy

Programové vybavení (software) je podmínkou využití technického vybavení (hardware). V raných dobách byly programy vytvářeny vždy pro konkrétní hardware. S masovou výrobou počítačů vznikla potřeba systémových programů, které zajišťovaly základní operace s hardwarem a poskytovaly prostředí pro běh aplikací.

Základním programovým vybavením počítačů jsou operační systémy (OS). Jejich nejdůležitější funkcí je řízení hardware prostřednictvím jádra systému, které spravuje procesor, paměť, disková média a další zařízení. Jádro pro své operace využívá ovladače (drivers).

Nezbytnou součástí OS je aplikační rozhraní (API), které poskytuje programátorům nástroje pro vývoj aplikací. Uživatelské rozhraní umožňuje ovládání OS a spouštění aplikací. Starší OS se často ovládaly příkazovým řádkem (CLI), moderní mají grafické uživatelské rozhraní (GUI) s okny a grafickými prvky.

Na počátku moderních OS byl UNIX, víceúlohový a víceuživatelský systém používaný od 70. let. Z jeho principů vycházely další systémy, například MS DOS (Microsoft) a Mac OS (Apple). Mac OS zavedl grafické rozhraní, které se stalo základem MS Windows. Od té doby se také rozšířil Linux.

Operační systémy dnes dominují na různých typech zařízení:

• Síťové servery (např. Windows Server, Linux, Unix)

• Desktopy a notebooky (Windows, Linux Ubuntu, OS X)

• Mobilní zařízení (Android, iOS, Windows Phone, Symbian)

• RTOS (Real-Time Operating System) – systémy reálného času pro průmyslové a specializované aplikace.

Zatímco desktopové OS umožňují paralelní operace, RTOS provádějí úlohy bez prodlevy a s vysokou spolehlivostí v reálném čase.
